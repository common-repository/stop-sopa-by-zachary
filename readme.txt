This plugin help you to set all page templates to sustain the couse 'Stop SOPA'.

== Description ==

The site will look dark and will have just one spot of light. This spot of light will move with the mouse cursor, and the message will be revealed.

== Installation ==

To install, simple extract the plugin files into your 'wp-content/plugins/' directory and activate the plugin.
